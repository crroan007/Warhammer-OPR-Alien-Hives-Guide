# Warhammer OPR — Alien Hives Battle Guide

A phone-friendly, single-file battle guide for a **1000 pt Alien Hives "Monster Mash"** army in
**One Page Rules: Grimdark Future (v3.5.1)**, built to beat a **Human Defense Force** gunline.

**▶ Live guide:** https://crroan007.github.io/Warhammer-OPR-Alien-Hives-Guide/

## Tabs
- **Each Round** — step-by-step actions to take every round (pre-game → R1–R4), with the *why* for each step, plus a full rules glossary.
- **Strategy** — pick any two goals and get the tailored game-plan for that combination (6 goals, all 21 combos).
- **My Army** — every unit: stats, what it does, how to use it (and what not to do).

## The one-line plan
Beasts soak the gunline up the middle (a single-model Carnivo-Rex caps any Blast weapon to 1 hit); fast units flank to kill the artillery and grab markers. **You win on objective markers, not kills.**

## Before you play — verify in Army Forge
- **Hive Bond** (your army rule) — exact effect.
- **Destructive** (big-beast weapon rule) — exact effect.
- Whether you're using the optional **Force Org** rules.
